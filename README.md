## 🔬 Project Overview
This project goes beyond standard tutorial-level classification. It treats the EuroSAT dataset (27,000 Sentinel-2 satellite images across 10 land-use classes) as a real-world research problem, culminating in an analysis of the "Semantic Noise Floor."

### Architecture Progression
1. **Baseline CNN:** Built from scratch. Achieved **77.83%** Macro-F1.
2. **ResNet18 (Transfer Learning):** Frozen feature extractor, fine-tuned classifier. Achieved **97.70%** Macro-F1.
3. **EfficientNet-B0 (Compound Scaling):** Tested for edge-device (drone) deployment. Achieved **93.11%** Macro-F1 with less than half the parameters of ResNet.

### Explainable AI (XAI) & Error Analysis
Using **Grad-CAM (Gradient-weighted Class Activation Mapping)**, we visualized what the ResNet18 model was actually looking at. We uncovered a critical "ImageNet Edge Bias"—the model was hyper-focusing on parallel lines. 

By running a hard-examples error analysis, we discovered that the model's confusion between the `River` and `Highway` classes was not a model failure, but a dataset failure. Ships, wakes, and artificial canals create perfect parallel lines in rivers that mimic paved roads, proving we had hit the **Semantic Noise Floor** of the dataset.
